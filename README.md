# Soundspace-custom-songs
all sspm files, I HAVE MADE NONE OF THESE!!!

At least in my case, using the itch.io soundspace+ when uploading custom maps, sometimes the application will bug out and crash. Afterwards, you'd try to upload it again, but it tells you 'this map already exists' even though you don't see it anywhere and it's unplayable. To fix this, in the application navigate to settings, and then 'user folder'. This opens the folder where all custom maps have been uploaded, so all the uploaded files should be there. Find the file for the map you uploaded unsuccesfully, and delete it from the folder. Then, from whatever file location you initially had the sspm saved to before uploading it to soundspace+, just copy and paste that into your user folder and restart the application. Then, you should be able to see and play the custom map! This can also work for uploading large quantities of custom maps all at once, as uploading them one at a time through the in-app method can be pretty tedious. Do note, though, that as far as I know, you can't edit any of the map information aside from the difficulty rating after you've uploaded a custom map, so if you want to change any of that, you're going to have to upload it using the built in method.
